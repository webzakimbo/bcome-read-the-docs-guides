******
Routes
******

The ``routes`` command displays the pathways your SSH connections will take to reach your servers.

The following AsciiCast demonstrates its usage:


.. raw:: html

   <a href="https://asciinema.org/a/pQRwqdJ50Uukkx7GWcgIr530J" target="_blank"><img src="https://asciinema.org/a/pQRwqdJ50Uukkx7GWcgIr530J.svg" /></a>

.. note::

   To replay this Asciicast in your own terminal, install the ``asciinema`` package from https://asciinema.org/, and then enter the following in your terminal:

   ``asciinema play https://asciinema.org/a/pQRwqdJ50Uukkx7GWcgIr530J``
